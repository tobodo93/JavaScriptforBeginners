You need to create a variable in JavaScript before using it, or, as it is called - declare.

The declaration of a variable is done using the keywords var or let. It looks like: 
var x;
var studentName;let x;
let studentName;

At the end of the line should be a semicolon, and this applies not only to declaring a variable, but to any operation in JavaScript. This is not a strict language requirement, but the absence of a closing semicolon can sometimes lead to unpredictable consequences.

You can assign a value to a variable, or initialize it at the declaration stage:

var x = 1;      
let studentName = "Vasiliy";

Initialization of an already declared variable must  be done without the var directive: 

x = 1;      
studentName = "Vasiliy";
