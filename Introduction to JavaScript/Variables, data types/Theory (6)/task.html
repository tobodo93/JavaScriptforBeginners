Now let's take a closer look at the process of solving programming problems that we will meet.

The task is simple - assign the input value from the variable "a", passed to our function, to the variable "x", which we return.

In the opened window of the editor, we see the preparation of the code. 

function testA(a) {      // Function definition. This is service information, you can not change it.
                         // We see that the function has an input parameter - the variable "a".
    var x="";            // The declaration of the variable "x", which we will return as a result.

                         // Here you need to write a solution - the code that will put the result in "x".

    return x;            // The command to output the result, it also undesirable to change.
}                        // The closing code is a curly brace, it must be present.

Also in the job, you will see the Sample Input and Sample Output expressions. It is an examples.
Sample Input shows what can be passed to our function as a variable "a"
Sample Output demonstrates that as a result you must assign the variable "x"


Since in next task we need to return the input data unchanged, the Sample Input and Sample Output parameters will be the same.
