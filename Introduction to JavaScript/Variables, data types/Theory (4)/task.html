Now, knowing what a variable is in JavaScript and knowing how to create it and write data to it, let's try to figure it out - what can we write into it at all? What types of data can store variables in JavaScript?

All data in JavaScript can be of the following types:
- Numbers
- Strings
- Logical (boolean)
- Undefined
- Object
- Empty


In order to determine the type of data written in a variable, we can use the typeof operator.

For example, here is a script: 

var myName = "Ivan";

alert (typeof myName);

will output the message "string";
