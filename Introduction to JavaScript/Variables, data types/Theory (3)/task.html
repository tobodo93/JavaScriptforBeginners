Variables in JavaScript can be Local and Global

Local variables are variables declared within a JavaScript function. They are only available within the function in which they are declared. When you exit this function, variables are destroyed.

You can declare variables with the same name inside different functions - they will not overlap in any way, because they are used only within the function in which they are created.

Global variables are declared outside functions and all functions and scripts can access to them. These variables are destroyed when the page is closed.

If you declare a variable without using the keyword " var", then it is automatically declared global. For example, the expressions x = 5; or surName = "Ivanov"; will declare variables x and surName as global if they do not already exist.
