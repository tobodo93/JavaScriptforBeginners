The declaration with the let keyword has several important differences from the declaration with the keyword var.

1. Scope.

Unlike the variable declared through the keyword var and visible inside the entire function in which the declaration occurred, the variable declared by let is visible only inside the {...} block in which it was declared.

2. Time visibility.

The variable declared by the keyword let is visible only after the declaration, and the variable declared by the keyword var can be available in the code up to the point in which it is declared, of course within the general rules of variable visibility.

3. When you use a variable as a loop counter, the variable declared through the word var lives for the duration of the loop and available even after it completes. If you use the declaration via let, each iteration of the loop will have its own independent variable. We will consider this use case in the third module, when studying closures.
