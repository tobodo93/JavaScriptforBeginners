Now let's look at each data type in more detail.

The type of data number is formed if any number is assigned as a value.

For example, the command var x = 5; will create a variable of type number.

And if in this example the number 5 is enclosed in quotation marks, (var x = "5";) the type of the variable will be string.

The type of the variable "String" is obtained if the value assigned to the variable is enclosed in single or double quotes.

The logical data type is boolean: these are just two variants of the value of the variable - true (true or logical 1) and false (false or Logical 0). This data type is used when applying comparison operators, logical operations, and branching operators.

The undefined variable has a type when it is declared but not yet initialized, so, it was created and the value has not been assigned yet.

Data types object and null we will consider later.
