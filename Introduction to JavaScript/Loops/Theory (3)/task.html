The for loop has one more kind - the for ... in construct
This operation searches through all the allowed properties of the object. In more detail, we will consider the use of this construct later, when studying objects, but for now, just mention its syntax: 


for (variable in object) {
   executable code
}

For example, the following code will display all the properties of the car object 

for (i in car) {
   console.log (car [i] + '');
}
