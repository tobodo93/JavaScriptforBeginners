The For loop is used in cases where we can determine in advance how many times a given block of code should be executed.

The command syntax looks like this: 

for (variable = start_value;
// the initial value of the counter variable is set firstvariable <= final_value;
// after this, a condition is set, which is checked during each loopvariable = variable + step)
// and finally the last one is the operation of changing the state of the variable-counter{
executable code
}


For example: 


for (let i = 1; i <= 5; i = i + 1) {
  console.log ("i =" + i);
}

The result of this loop will be five times deducing the value of the variable i in the form "i = 1", "i = 2", etc.
