The while loop has a variation: the do ... while construct
This design differs from the previous one in that the condition check operator is located AFTER the main body of the executable code, which ensures at least one execution of the code block.

The syntax is as follows: 

do {
  executable code
} while (variable == value);

Here is an example of using this construction: 

var x = 5;                     // set the initial value
do {                           // start the loop
  console.log ("x =" + x);     // output data to the console
  x = x - 1;                   // decrease the value by 1
} while (x> = 1);              // check condition

In this case, the program will output to the console the value of the variable "x" in the range from 5 to 1, the result will be "54321"
