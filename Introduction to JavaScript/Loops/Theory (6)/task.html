We'll finish the lesson about loops by examining two commands that provide an interruption to the loop: break and continue.

The keyword break allows us to abort the loop and go to the code following the loop.
Let's consider an example: 


for (i = 1; i <10; i = i + 1) {
   if (i == 5) {
     break;
   }
   console.log ('i =' + i);
}

In this case, the loop is interrupted when the variable i reaches the value of 5. Up to this point, each iteration will result in the console outputting the values of the variable i.
