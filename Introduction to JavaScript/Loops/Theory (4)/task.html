Now consider the while loop. It executes the block of code as long as the specified condition is true.
The syntax is as follows: 


while (variable == final value) {
// of course, any comparison operator can be used in the loop, not only ==
   executable code
}

For example : 

var x = 1;
// declare and initialize a variable that
// we will use as the condition of the loop
while (x <= 5) {
   console.log ("x =" + x);
   x = x + 1;
}

As a result of this code, the value of x will be output five times.
