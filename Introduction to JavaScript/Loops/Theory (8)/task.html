Speaking of loops and exit from them, you should also mention such a tool as labels . Labels in JavaScript only work for loops and the switch statement, so that they can not make a full transition to any point in the program (like "goto" loops in other programming languages). You can use a label to identify a loop, and then use the break or continue statements to indicate whether a program should interrupt the loop or continue its execution.

For example: 

label_1: 
for (i = 1; i <= 50; i ++) {
  while (i < a) {
    console.log (i);
    if (i == a) {
      break label_1;
    }

    if (i == b) {
      continue label_1;
    }
  }
}


In this example, if the i value is equal to the value of the variable "a", the outer for loop will be interrupted, and if the i value is equal to the value of the variable "b", it will exit the current iteration and continue the work of the outer loop.
