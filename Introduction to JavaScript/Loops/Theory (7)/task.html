The continue keyword terminates the current iteration of the loop and proceeds to the next.
As an example, consider this code: 


for (i = 1; i <10; i = i + 1) {
   if (i == 5) {
     continue;
   }
   console.log ('i =' + i);
}

In this case, continue will prevent the output to the console from being executed with the value of the variable equal to 5, all other values will be printed.
