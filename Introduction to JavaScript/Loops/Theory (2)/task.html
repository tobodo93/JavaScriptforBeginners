Let's now take a closer look at the work of the FOR loop.

for (i = 1; i <3; i = i + 1) {
  console.log ("i =" + i);
}


In the first line, we set the parameters of the loop:

i = 1 // Set the value from which the account will start.
i <3 // We specify a condition that will be checked for each loop (usually say - every iteration)
i = i + 1 // Specify how the counter will change at each next iteration - increase by 1


In curly braces, we write the body of the cycle - the code that will be executed with every repetition of the loop, while our condition is true.

{
  console.log ("i =" + i); // The body of loop. In our case - output to the console of the text "i =" and the value of the variable i
}



In this example, the following actions occur:

Set the initial value
1. The variable i gets the value "1"

The first iteration
2. The condition is checked that i is less than 3
3. Since i is now 1, i.e. less than 3, then the loop body is executed, in our example, the variable i is displayed.
4. The variable i increases by 1 and its value becomes 2.

The second iteration
5. The condition is checked that i is less than 3
6. Since i is 2, i.e. less than 3, then the loop body is executed - the variable i is displayed.
7. The variable i increases by 1 and its value becomes 3

The third iteration
8. The condition is checked that i is less than 3
9. Since i is 3, i.e. the condition i <3 ceases to be true, the work of the loop is completed.
