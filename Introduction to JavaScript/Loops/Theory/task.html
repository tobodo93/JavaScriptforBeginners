Very often we have to repeat the same block of code to perform the same operation with the same variables but with different values. In order not to write this code block every time completely, we can use Loops.

Loops execute the necessary code block the required number of times, depending on the state of the counter or on the specified condition.

In JavaScript, we have two types of loops :
for - execution of the code specified number of times
while - execution of the code until the condition is true.
