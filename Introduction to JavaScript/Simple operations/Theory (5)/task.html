The operator ! (Logical NOT)
! (x == y) is true


JavaScript also has a so-called conditional operator (it is also sometimes called a ternary operator), it assigns a value to a variable based on a certain condition. The syntax of the conditional statement is: 

variable_name = (condition)? value1: value2;


For example, expression 

hiJack = (friend == "Jack")? "Hi, Jack!" : "Hi";

will lead to " Hi, Jack" output if friend variable will be " Jack" or " Hi" at any other value.
