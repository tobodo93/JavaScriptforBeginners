Assignment operators are as follows:

In the examples we will consider x = 10; y = 5;

Operator = is a normal assignment operator. Executing x = y will result in x = 5.

Some arithmetic operators can be used with the assignment operator:

Operator += This is assignment with addition
Execution x += y will result in x = 15, equivalent to x = x + y

Operator -= This is assignment with subtraction
Execution x -= y will lead to x = 5, is equivalent to x = x - y

Operator *= Assignment with multiplication
Execution x *= y will result in x = 50, is equivalent to x = x * y

Operator /= Assignment and division.
Operation x/=y will result to x = 2, is equivalent to x = x / y

Operator %= Assignment with "remainder of division"
operation x %= y will result in x = 0, is equivalent to x = x% y
