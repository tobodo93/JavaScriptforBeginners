Let's start with the simplest ones - Arithmetic operators.
All the results of examples will be considered with the initial data y = 5

Adding  "+"
expression x = y + 2 yields the result x = 7; y = 5;

Subtraction "-"
the expression x = y - 2 gives the result x = 3; y = 5;

Multiplying  "*"
expression x = y * 2 yields the result x = 10; y = 5;

Division "/"
expression x = y / 2 gives the result x = 2.5; y = 5;
