Comparison operators:

All the following examples are considered for x = 5

The operator is equal  "=="
This operation is performed with type conversion, i.e. if one of the operands is a number three and the second is a triple symbol, then the expression 3 == '3' will be true, since the second operand will be cast to the type of the first if it is allowed.
(x == 8 is false, x == y is true for y = '5')

The operator is strictly equal to  "==="
This comparison will be true without taking into account the type conversion
(x === 5 is true, x === '5' is false)

The operator is not equal to "!="
(x! = 8 is true)

The operator is not strictly equal to "!=="
(returns true if the operands are not strictly equal or have a different type.)

The operator is greater than ">"
(x> 8 is false)

The operator less than "<"
(x <8 is true)

The operator is greater than or equal to ">="
(x> = 8 is false)

The operator is less than or equal to "<="
(x <= 8 is true)
