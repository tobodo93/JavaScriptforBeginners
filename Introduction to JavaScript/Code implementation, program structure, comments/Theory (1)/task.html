Now let's consider an example of placing JavaScript code in a separate file.

We will create a file called myscript.js and put our code into it:

alert("Hello World!");

Calling the code from the body of the main HTML file will now look like this:

<html>
  <head>
    <title>Code sample page with JavaScript</title> 
    <script src="myscript.js"></script>
  </head> 
  <body>
    Text of the main page
  </body> 
</html>

The advantages are obvious: any amount of JavaScript code can be called by one small command.
