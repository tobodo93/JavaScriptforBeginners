You can implement JavaScript in an HTML document in two ways:
The first is to place the code directly in HTML file
The second is to place the code in a separate file.

Let's consider both ways:
When you place code inside an HTML file, the JavaScript code is framed with a <script> tag
It looks like this:

<script>

   ... JavaScript-code ...

</script>

 

Let's see what the code looks like on this page:

<html>
 <head>
  <title>Code sample page with JavaScript</title>
  <script>
   alert("Hello World!");
  </script>
 </head>

 <body>
  This is the main page text
 </body>
</html>

 

When you open this page, a window will appear with the text "Hello World!" and with "OK" button. After clicking on the "OK" button, the page will continue to run and you will see the text "This is the main page text".
