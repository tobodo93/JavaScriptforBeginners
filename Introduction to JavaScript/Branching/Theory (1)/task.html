There is another way to use this operator - when you can execute some code if the condition is true and another code if the condition is false. The syntax is as follows: 

if (condition) {
code to execute if the condition is true
} else {
code to execute if the condition is false
}


As an example, consider this option: 
var x = 10;
var y = 50;

if (x > 11) {
   x += y;
} else {
   x -= y;
}

In this case, if x>11, then it increases by y, and otherwise it decreases by the same amount.
