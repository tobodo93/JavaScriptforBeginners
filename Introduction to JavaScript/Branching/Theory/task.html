In this lesson we will consider options for changing the behavior of the program depending on the values ​​of the given condition.
These constructions are called conditional statements.
In JavaScript, there are two possible conditional statements: if ... else construct and switch

The first one is used to execute some code if the specified condition is true. 
if (condition) {
code to execute if the condition is true
}


Let's look at an example: 
var x = 10;
var y = 50;

if (x > 11) {
 x += y;
}


In this example, the operator checks the condition and if x is greater than 11, then the variable x is incremented by y.
