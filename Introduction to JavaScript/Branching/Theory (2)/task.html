It should also be noted that if ... else statements can continue each other. For example: 

if (x> 11) {
   x += y;
} else if (x == 4) {
   x -= y;
} else {
   x = y;
}


In this example, x increments the value by y if it is greater than 11, decreases the value by a value of y if it is 4, and takes the value of y in all other cases, i.e. if x <11 and not equal to 4.
