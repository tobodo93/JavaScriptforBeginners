Object properties are some values associated with this object.

The syntax for accessing the object's property is as follows: 
objectName.propertyName



For example, consider a String object. It has a length property, which specifies the number of characters in a string. To access this property, we use the property name via a dot. For example: 


var textString = "Hello world!";
console.log (textString.length);

As a result of this code execution, the number of characters in this line will be displayed in the output console - 12.
