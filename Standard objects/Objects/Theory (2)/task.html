Methods are actions that can be performed on an object. They may be present at the object, and may not be present.

The syntax for accessing the method is as follows: 

objectName.methodName();


(In parentheses, the parameters to be passed to the method are specified.)

For example, a String object has a method called toUpperCase() that returns the contents of our object, that is, text, in capital letters. In particular, the code execution: 


var textString = "Hello world!";
console.log (textString.toUpperCase ());

will print to the console the text " HELLO WORLD!" (all letters will be uppercase).
