To create an object with constructor, we need to describe a function that will create an object and assign values ​​to its properties. The function will look like this: 

function Person (name, age, year) {
  this.name = name;
  this.age = age;
  this.year = year;
}


Inside the function, we assign the data passed to the function through the expression this - accessing the current instance of the object. (We'll discuss this in more detail at the end of the lesson).

Creating an instance of an object using our constructor is done as follows: 

var employee1 = new Person ("Ivan", "25", "2017");
var employee2 = new Person ("Olga", "21", "2016");
var employee3 = new Person ("Oleg", "32", "2010");

It is important to note that the name of the constructor is usually written with a capital letter to mark that it is a constructor and it must be called with the keyword new. Otherwise, this in the body of the constructor will point to anything, not just the created object.
