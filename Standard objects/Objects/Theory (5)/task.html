To add a new object property, there are also two options. 

person.name = "Ivan"

or 

person ['name'] = "Ivan"

You can also add properties immediately when creating an object by specifying a list of properties in curly braces: 

var person = {
name: "Ivan",
age: 25,
hiredYear: 2017
}

Accordingly, to access this property, you can also use two options:
objectName.PropertyName, for example: 
console.log (person.name);

or objectName[propertyName], for example: 
console.log (person['name'])

If we try to access a property that the object does not have, then the result will be 'undefined', the so-called "undefined value".

To delete a property, use the delete operator: 

delete person.name;
