Conversion and comparison functions:

Math.min ([Value1 [, value2 [, ...]]]) - Returns the minimum value of the arguments.
Math.max ([Value1 [, value2 [, ...]]]) - Returns the maximum value of the arguments.
Math.floor (x) - Returns the largest integer less than or equal to the argument
Math.ceil (x) - returns the smallest integer greater than or equal to the argument
Math.abs (x) - returns the absolute value of a number, it is also called a "module"
Math.round (x) - rounds the number according to the rules of mathematics
