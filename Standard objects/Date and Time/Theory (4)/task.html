Write data to a Date object.

setYear() - sets the value of the year minus 1900, not recommended for usage.
setFullYear() - sets the value of the year.
setMonth() - sets the month, from 0 to 11
setDate() - sets the day of the month from 1 to 31
setHours() - sets the hour, from 0 to 23
setMinutes() - sets the number of minutes from 0 to 59
setSeconds() - sets the number of seconds from 0 to 59
setMilliseconds() - sets the number of milliseconds, from 0 to 999
setTime() - sets the number of milliseconds that have elapsed since midnight on January 1, 1970 GMT.
