Functions in JavaScript are blocks of code that have their own name (except anonymous ones) and can be called by that name.

Functions in JavaScript are built-in, such as alert() and user-defined that the programmer creates by himself.

In Functions in JavaScript, some parameters can be passed, for example, those necessary to calculate the return values.

Functions in JavaScript are objects, and as a consequence they can, for example, be assigned to variables, passed to other functions, you can assign values ​​to their properties and call their methods. (We'll discuss the concept of the object in more detail in the next lesson).

Functions in JavaScript are named by the same rules as variables.

The main purpose of the function is to save the program from duplicating the code. By writing a certain sequence of commands, you can call it as many times as you like.
