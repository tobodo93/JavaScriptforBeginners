Creation, or function definition begins with the function keyword, followed by the following data:

Name of the function to be used when creating the variable to which the object of our new function will be assigned.
List of input parameters in parentheses (may be absent).
Body of the function in curly braces is the actual list of executable commands (also may be missing).

Let's look at an example of creating a function that will output the words "Hello World!" 

function printText () {             // Function definition
  document.write ("Hello World");   // Function body - output text to document
};

Now, in order to print on the screen the words "Hello World!" we just need to call this function in the code: 

printText ();
