In this lesson we will consider working with string objects.

You can create a String object in three ways:

  • myString = 'my string'                 // creating a string object using a string literal
  • myString = new String (object) // creating an object, such syntax is not used anymore
  • myString = String (object)

Such objects are typically created implicitly, by using a string literal: var myString = 'Some text';
(a string literal is a sequence of characters enclosed in any kind of quotes)
In a string literal, you can use any characters as well as escape sequences, for example, if you need to specify a character that can not be typed on the keyboard. It looks as follows: var myString = '\u1234'; where 1234 is the number of the character in the unicode table.

An important point! When you create an object using a literal (text in quotation marks), the resulting type is "string", and when you create an object using the keyword "new", you get an object with the "Object" type, which can be assigned additional properties and methods directly.
