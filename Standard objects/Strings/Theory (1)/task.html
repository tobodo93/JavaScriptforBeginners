Properties of the "String" object

Although, of course, the word "properties" sounds too loud - the standard property of it is one - length. It is denoted by the word ". length" and returns the number of characters in the string. 


var cat = "Cat";
console.log (cat.length);

In this example, the value of the string length is 3.

The value of the length of the empty string, of course, is 0.
