For example, to use an object in a function from the previous example, we would need to write like this: 

// Declaration of the function
function testFactorial (inputData) {
  if (inputData <0)                                            // Check if the number is positive
    throw new Error ("The number must not be less than zero"); // Create and throw an Error object
  return (inputData - 1)? (inputData * testFactorial (inputData - 1)): inputData;
}

var myNumber = -5;

try {
    document.write (testFactorial (myNumber));                  // attempt to call a function
} catch (ex) {
  document.write (ex.message);                                  // output error message
}

In this case, we intercepted the exception and output its message property into the document, to which we previously placed the text - " The number should not be less than zero".
