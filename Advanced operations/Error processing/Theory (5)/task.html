Also we need to consider the standard Error object, which is used when using exceptions.

In general, almost any object can be thrown by the throw command, however, when standard errors occur, the object of the Error class or its subclasses is thrown out.

The syntax that we will look at is as follows: 

new Error (message);

This command creates a new instance of the Error object, and the text " message" is written to the property "message" of the object.

The simplest example of calling the Error constructor is as follows. 

try {
  throw new Error ('Something went wrong!');
} catch (e) {
  console.log (e.name + ':' + e.message);
}


In this example, we created an Error object with the name Error and the text of the error message " Something went wrong!".
